# Bot-Discordpy - Bot discord orienté HomeLab

## Description : 
Bot discord en cours de développement conçu pour être utilisé dans un HomeLab, permettant de gérer diverses tâches et automatisations. Si vous avez des idées de fonctionnalités ou souhaitez contribuer, n'hésitez pas à me contacter ! Discord : nahnah_0

## Fonctionnalités :
- **Téléchargement de vidéos** : Permet de télécharger des vidéos à partir de différentes plateformes (TikTok, YouTube, etc.).
- **Gestion de tâches** : Permet de créer, suivre et gérer des tâches au sein du HomeLab. (En cours de développement)
- **Rappel de tâches** : Envoie des rappels programmés à l'utilisateur.

## Installation :
1. Clonez le dépôt :
   ```bash
   https://github.com/Neyz1/Bot-Discordpy.git
    ```
2. Installez les dépendances :
   ```bash
   pip install -r requirements.txt
   ```
3. Configurez le bot en ajoutant votre token Discord dans le fichier de configuration.
   ```
   echo "DISCORD_TOKEN=your_token_here" > .env
   ```
4. Lancez le bot :
   ```bash
   bash inti.sh
   bash start.sh
   ```
5. Le bot devrait maintenant être en ligne et prêt à être utilisé !
